分层架构。
稳如磐石。

1

App / Widget 层：基于 Qt 构建的高级交互逻辑与 UI 展现。

2

Service / Runtime 层：工厂职责，动态管理音视频与网络核心模块。

3

Backend / API 层：极致解耦。支持 WebRTC、FFmpeg 多后端切换。
